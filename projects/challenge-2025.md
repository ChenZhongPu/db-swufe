## 题目三

考虑下面的题目，

```
考虑create table classroom
 (building  varchar(15),
  room_number  varchar(7),
  capacity  numeric(4,0),
  primary key (building, room_number)
 );找出容量最大的教室房间号
```

其答案是

```sql
SELECT room_number
FROM classroom
WHERE capacity = (SELECT MAX(capacity) FROM classroom);
```
