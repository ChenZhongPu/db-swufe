\begin{frame}
    \frametitle{为什么需要学习关系代数}    
虽然商业数据库基本都是采用 SQL，而不是直接使用关系代数，但是当 DBMS 执行查询的时候，它做的第一件事就是\textbf{将 SQL 翻译成关系代数（或者其他类似的表示）}。

\pause

关系代数有用的第二个原因是：\textbf{相较其他通用的编程语言（如C、Java），它不够强大}。它仅提供有限的操作，从而便于查询编译器能够产生优化的代码。

\end{frame}
